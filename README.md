# ETL Pipeline with Python, SQL, and Airflow

## Overview
This project extracts data from a public API, transforms it into a structured format, and loads it into PostgreSQL.

## Tech Stack
- Python
- PostgreSQL
- Pandas
- Requests
- SQLAlchemy

## Pipeline Steps
1. Extract data from API
2. Save raw data locally
3. Transform and clean the data
4. Load processed data into PostgreSQL
